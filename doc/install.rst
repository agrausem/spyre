===============================
Installing `spyre`
===============================

.. _install_from_github:

Installing from GitHub
======================

    git clone git://github.com/bl0b/spyre.git
    cd spyre
    virtualenv env
    source virtualenv/bin/activate
    pip install -r requirements.txt
    pip install -r requirements-test.txt

.. _testing:

Testing
=======

    ./run_tests.py
